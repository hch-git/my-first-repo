 本周学习收获

1. 仓库基础操作

◦ 学会了用 GitHub Desktop 可视化操作仓库，也了解了 git add、git commit、git push 等核心命令的作用。

◦ 掌握了如何关联本地仓库与 GitHub 远程仓库，解决了网络连接问题。

2. 分支与冲突管理

◦ 理解了分支的概念，学会了创建、切换、合并分支的完整流程。

◦ 亲身体验了合并冲突的场景，并学会了如何手动解决冲突。

3. 文档与协作规范

◦ 掌握了 Markdown 的基础语法，能写出结构清晰的 README.md。

◦ 理解了 .gitignore 文件的作用，学会了配置忽略规则。

◦ 养成了规范的提交信息习惯，让提交历史清晰易读。

 踩过的坑 & 解决方案
遇到的问题 解决方法 
克隆/推送仓库失败，无法连接 GitHub 修改系统 hosts 文件，刷新 DNS 缓存解决网络问题 
合并分支时出现冲突 在编辑器中手动合并冲突内容，保留双方需要的修改后再提交 
Windows 系统找不到 .gitignore 文件 开启文件资源管理器的「显示隐藏文件」选项
