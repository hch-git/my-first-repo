## 项目简介
这是我在两周 Git 实践中创建的第一个仓库，用于联系分支管理，冲突解决，Markdown 文档编写等基础技能

## 功能特性
- 基础 Git 分支创建与合并
- 规范使用 README 文档编写
- 冲突解决流程演练

## 安装步骤
1.  克隆仓库
    ```bush
    git clone https://github.com/你的用户名/my-first-repo.git
    cd my-first-repo
    ```


2.  查看分支列表
    ```bush
    git branch -a
    ```
